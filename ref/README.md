# Open-source references

Files in this directory are retained only for comparison and design review.
Runtime code must not import them.

- `ddt_tita/`: selected BSD-3-Clause Tita task configurations from DDT_Lab.
- `mujoco_playground/`: upstream project documentation and disabled release
  workflow retained from the original MuJoCo Playground layout.

The exact source revisions and repository URLs are recorded in
`../third_party/versions.yaml`. Preserve original copyright and license headers
when adding files here.
