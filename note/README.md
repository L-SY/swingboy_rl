# Swingboy RL Notes

Interactive project notebook for robot-model validation, MDP design review,
reference-repository comparison, and experiment tracking.

```bash
cd note
npm install
npm run dev
```

Checklist state, edited baseline values, decisions, and experiments are stored
in browser `localStorage`. Use the JSON export action before clearing browser
data or moving to another computer.

The visual language is inspired by `L-SY/personal_ui`, but this implementation
is original because that reference repository does not currently provide a
reuse license.
